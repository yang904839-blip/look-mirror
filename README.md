# LookMirror

一个简洁的 macOS 菜单栏应用，让您快速检查仪容仪表。

## 功能特性

- 🪞 **快速镜像**：点击菜单栏图标即可查看摄像头画面
- 🔄 **镜像切换**：支持镜像/正常模式切换（默认镜像）
- 📐 **多种画幅**：支持 16:9、9:16、3:4、4:3、1:1 五种画幅
- 🎨 **系统特效**：快速访问 macOS 系统视频特效面板
- 🔒 **隐私保护**：仅在您主动打开时使用摄像头

## 安装方法

1. 打开 `LookMirror-1.0.dmg`
2. 将 `LookMirror.app` 拖到 `Applications` 文件夹
3. 从启动台或应用程序文件夹打开 LookMirror
4. 首次运行时允许摄像头权限

## 使用说明

1. **启动应用**：应用会在菜单栏显示一个胡子图标 🥸
2. **查看镜像**：点击菜单栏图标，弹出摄像头画面
3. **打开设置**：点击画面左上角的齿轮图标
4. **调整设置**：
   - **Mirror My Video**：切换镜像/正常模式
   - **Aspect Ratio**：选择画幅比例
   - **System Panel**：打开系统视频特效设置
   - **Quit**：退出应用

## 系统要求

- macOS 13.0 (Ventura) 或更高版本
- 内置或外接摄像头

## 隐私说明

LookMirror 仅在您点击菜单栏图标时启动摄像头，不会在后台录制或上传任何数据。所有处理均在本地完成。

## 开源

本项目使用 Swift 和 SwiftUI 开发，采用 Swift Package Manager 构建。
